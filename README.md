# E-commerce Funnel & Conversion Analysis

This project analyzes over 500,000 user interactions from an online cosmetics e-commerce dataset.

## Key Findings
- High visibility does not reliably translate to higher conversion
- Regression (R² = 0.0845, p = 0.335) confirms weak relationship
- Significant drop-off before purchase completion

## Tools
R • tidyverse • ggplot2 • Linear regression
